## What this project provides

- Tool definition standard
- Runtime execution layer
- Context abstraction
- Capability routing
- Workflow orchestration
- Provider adapters
- Guardrail middleware
- Trace events and OpenTelemetry export

## First principles

AI application complexity comes from integration, not from calling a model.
This toolkit standardizes the integration layer.

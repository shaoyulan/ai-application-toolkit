# AI Application Toolkit

Composable infrastructure for AI applications.

## What this project provides

- Tool definition standard
- Runtime execution layer
- Context abstraction
- Capability routing
- Workflow orchestration
- Provider adapters
- Guardrail middleware
- Trace events
- npm release pipeline
- Nuxt documentation site

## First principles

AI application complexity comes from integration, not from calling a model.

This toolkit standardizes the integration layer.
